BloggersInc.

HTML5 Javascript Node.JS Semantic UI MongoDB CSS

This is one of my favourite websites I have created so far. MongoDB was used to store user data and create links between different sets of data. Passport JS library was used to create a safe login form and authenticate users safely. The website also involes restful routing, allows user to create or edit their blogs, and make or edit their comments. A functional search bar was made using mongoose and express libraries. MongoDB allowed user profiles to be created and Semantic UI was used for styling the profile cards,which can be viewed,users can also view other authors' profiles, and see how many blogs that particular author created.

This site allows users to edit and create campgrounds, give reviews, and also edit and create comments on the campgrounds,all this data are stored in non relational database created by MongoDB. GameCon

HTML5 Javascript Node.JS Bootstrap4 MongoDB CSS

This is a game review website that I have created. It allows users to add new games, add reviews and edit them, and also allow the users to make or edit their comments on the games. The website involves passportJS for user authentication and MongoDB was used to create and store non relational databases. The website requires little backend work, I also implemented Express libary which allowed me to easily send HTML requests to the web server.
Prosthetic Hand

AutoDesk Inventor

The aim of this project was to make a fully functional gear train, that will move the thumb and the index finger at a certain output speed with a certain input speed. I worked in group of 3 to get the job done. I did the preliminary sketches, and almost all of the drawings, we made in this project. I used AutoDesk Inventor to CAD some of the spur gears, and made some gear calculations too. We were successful in getting the design to work. This project demonstrates my ability to work with others, coming up with really good ideas, and also my abilities of CAD.
